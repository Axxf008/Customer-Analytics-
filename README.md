# 📊 Customer Analytics SQL Project

## 🔹 Overview

This project demonstrates SQL skills by analyzing a customer dataset
imported from Excel into MySQL.

## 🔹 Features

-   Data import from CSV using `LOAD DATA`
-   Data cleaning and structuring
-   Analytical queries for business insights
-   Customer segmentation

## 🔹 Tech Stack

-   MySQL
-   SQL
-   Excel (Data Source)

## 🔹 Dataset

-   Source: Excel file converted to CSV
-   Contains customer demographics and membership data

## 🔹 Key Insights

-   Customer distribution by city
-   Membership type analysis
-   Age-based segmentation

## 🔹 How to Run

1.  Create database:

``` sql
SOURCE sql/01_create_database.sql;
```

2.  Create tables:

``` sql
SOURCE sql/02_create_tables.sql;
```

3.  Load data:

``` sql
SOURCE sql/03_load_data.sql;
```

4.  Run queries:

``` sql
SOURCE sql/04_basic_queries.sql;
SOURCE sql/05_advanced_queries.sql;
```
